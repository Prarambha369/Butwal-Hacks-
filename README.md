## About Us

Butwal Hacks is a student-led community where curiosity meets creativity. We create spaces for young minds to experiment, collaborate, and transform ideas into reality across Rupandehi District, Nepal.

In Rupandehi District, access to tech events and creative communities is still developing. Many students have talent and passion but lack opportunities to practice skills in coding, design, engineering, robotics, and digital creativity. Butwal Hacks fills that gap with a platform that is open, inclusive, supportive, and driven by students who understand other students.

## ğŸš€ Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm, yarn, or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Butwal-Hacks-.git
   cd Butwal-Hacks-
   ```

2. Navigate to the Next.js app:
   ```bash
   cd my-app
   ```

3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

### Building for Production

```bash
npm run build
npm start
```

### Deploying to Vercel

The Next.js application is located in the `my-app` directory. The repository includes a `vercel.json` configuration file at the root that tells Vercel how to build and deploy the app from the subdirectory.

1. Push your code to GitHub
2. Connect your repository to [Vercel](https://vercel.com)
3. Vercel will automatically use the `vercel.json` configuration to build from the `my-app` directory
4. Click "Deploy" and your site will be live!

Alternatively, you can deploy directly using the Vercel CLI:
```bash
npm i -g vercel
vercel
```
